1. HTML基础结构

HTML文档的基本结构包括以下部分：

<!DOCTYPE html> <html lang="zh-CN"> <head> <meta charset="UTF-8"> <title>页面标题</title> </head> <body> <!-- 页面内容 --> </body> </html>

说明：

  • <!DOCTYPE html>：告诉浏览器这是HTML5文档，必须放在第一行
  • <html lang="zh-CN">：根标签，所有内容都包裹在其中，lang属性指定语言（有利于SEO）
  • <head>：包含元信息（如字符编码、标题），不显示在页面上
  • <body>：包含页面可见内容（如标题、段落、图片）

2. 常用标签（必学）

以下是HTML中最常用的标签，必须掌握：

2.1 标题标签（<h1>-<h6>）

<h1>一级标题（最大）</h1> <h2>二级标题</h2> <h3>三级标题</h3>

效果预览：

一级标题（最大）

二级标题

三级标题

说明：<h1>是页面主标题（一个页面建议只写1个），<h6>是最小的标题，字体大小依次递减。

2.2 段落标签（<p>）

<p>这是一个段落，用来写正文内容。</p> <p>段落之间会有默认的间距，让文字更易读。</p>

效果预览：

这是一个段落，用来写正文内容。

段落之间会有默认的间距，让文字更易读。

说明：<p>标签用来包裹正文段落，浏览器会自动在段落前后添加空白。

3. 语义化标签（推荐）

语义化标签是HTML5新增的，用来描述内容的含义（如<article>表示文章、<nav>表示导航），有利于SEO和屏幕阅读器。

<header> <div class="logo">前端教程</div> <nav> <ul> <li><a href="#">首页</a></li> <li><a href="#">教程</a></li> </ul> </nav> </header> <article> <h2>文章标题</h2> <p>文章内容...</p> </article> <footer> <p>© 2024 前端教程. 所有权利保留.</p> </footer>

说明：语义化标签不会改变页面样式，但能让代码更易读、更符合标准。
